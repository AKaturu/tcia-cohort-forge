# Security Policy

Please do not include protected health information, credentials, API tokens, private download manifests, or institution-specific data in public issues, pull requests, logs, or sample files.

To report a security concern, open a private advisory on GitHub or contact the repository owner directly. Provide the affected version or commit, a concise reproduction path using public or synthetic data when possible, and the expected impact.

This project is a research prototype and is not intended for independent clinical decision-making.
